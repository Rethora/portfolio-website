## About

An advanced NPM package designed for efficient recipe scraping from supported websites. This versatile tool not only allows you to scrape individual recipes but also empowers you to extract multiple recipes from a single URL containing links to various culinary delights. With built-in filtering capabilities, you can tailor results to match your specific criteria. Moreover, the package supports websites with pagination, enabling the extraction of results from a specified number of pages for a comprehensive and customizable recipe gathering experience.
